Municipalitatea unui oraș dorește să gestioneze detaliat informațiile legate de spațiile verzi, cartiere, mobilier urban, copaci, ape (lacuri, râuri), alei, și starea mobilierului urban. Fiecare oraș este împărțit în cartiere, iar fiecare cartier poate conține mai multe spații verzi. Aceste spații verzi pot conține copaci, mobilier urban (cu materiale și tipuri specifice), alei, dar și corpuri de apă sub formă de linii sau poligoane. De asemenea, se monitorizează starea mobilierului urban.

Entitati:
- oras
- cartiere
- spatii verzi
- apa, apa_polygon, apa_linie
- alee
- mobilier urban, stare mobilier urban
- copaci
